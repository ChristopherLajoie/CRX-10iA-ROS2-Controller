# CRX-10iA ROS2 MoveIt2 Controller

This is the repository for the controller I developped during my 4th internship at Michelin, Magog. It was used to demonstrate a FANUC CRX-10iA avoiding obstacles in its environment when
moving to its home position, using the FANUC ROS2 driver and MoveIt2 motion library.
